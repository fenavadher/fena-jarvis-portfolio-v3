# Fena Vadher — JARVIS AI Portfolio

Ultra-premium cinematic portfolio with JARVIS AI Interface theme.

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion

## Features

- Cinematic auto-play intro with particles & typing animation
- Profile intro with center→left image transition
- Interactive dashboard with 7 modules
- DSA visualizations: Sorting, Stack, Queue, Graph, Linked List, HashMap Terminal
- Glassmorphism UI with neon cyan/purple glow
